Caculate PIE using javascript
=============================

Calculating PIE based on [Ken Ward's formula](http://www.trans4mind.com/personal_development/JavaScript/longnumPiMachin.htm)

```
calculatePI(numberOfDigits);
```

## Usage / Examples

```
calculatePI(10000);

calculatePI(50000);
```
